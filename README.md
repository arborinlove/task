## bash 명령어

 pwd : 현재 디렉토리 위치

 clear : 현재 콘솔창으로 보여진 입력한 것들과 출력된 것들을 화면에서 지움

 touch 파일명 : 파일 생성

 cd /c : c 드라이브로 바로 이동

 date : 현재 시각
