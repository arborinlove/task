## bash 명령어

 pwd : 현재 디렉토리 위치

 clear : 현재 콘솔창으로 보여진 입력한 것들과 출력된 것들을 화면에서 지움

 touch 파일명 : 파일 생성

 cd /c : c 드라이브로 바로 이동

 date : 현재 시각

## git 명령어

 repository : 기록공간

 stage : 임시공간

 branch : 기준 작업 라인

 commit : 타임캡슐에다 넣고 커밋을 넣으면 소스가 쌓임. 언제든 과거로 돌아갈 수 있음

 switch 브랜치명 : 해당 브랜치로 이동

 restore 파일명 : 파일 복원
